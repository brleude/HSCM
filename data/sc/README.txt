Key and Non-key Proteins.xls：Saccharomyces cerevisiae key and non-key protein data.
The table contains two worksheets, storing key proteins and non-key proteins respectively. The data format is as follows：
YAL001C
YAL003W
YAL012W
YAL025C
YAL032C
......


PPI Network.txt：Saccharomyces cerevisiae PPI network data.
The data format is as follows：
Q0050 YER142C
Q0080 YBL099W
Q0080 YJR121W
Q0085 YBL099W
Q0085 YDR298C
......
Each row of data indicates that the two proteins in the row have an interaction.
